1.3.4 Web Technologies

This module will teach you the basics of how to create your own web pages.

The first couple of lessons are on HTML, CSS and Javascript.

A greate resource while you learn is W3Schools


Theme:



Lesson 1 - HTML

Objectives:

  • Understand what a HTML document is
  • Be able to create HTML documents
  • Be able to use a number of HTML tags and evaluate what they do

Outcomes:

  1. HTML tag definitions
  2. Example HTML code




Lesson 2 - CSS

Objectives:

  • Understand what a CSS document is
  • Be able to create CSS documents
  • Be able to use a number of CSS selectors and a number of CSS properties

Outcomes:

  1. CSS property definitions
  2. Example CSS code with both internal & external style sheets




Lesson 3 - Javascript

Objectives:

  • Understand why Javascript is used
  • Define basic javascript technology

Outcomes:

  1. Create a Javascript form
  2. Perform form validation
